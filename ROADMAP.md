# Dunia AI - Roadmap v0.2.0

## Vision

Faire de Dunia AI un :

- assistant IA personnel ;
- système multi-agents ;
- copilote de productivité ;
- outil de recherche ;
- plateforme d'automatisation locale.

## Objectifs principaux

### 1. Mémoire avancée

Fonctions prévues :

- mémoire long terme ;
- recherche conversationnelle ;
- interface de gestion memoire locale `/memory` ;
- tags intelligents ;
- embeddings locaux ;
- résumé automatique de l'historique.

Technologies envisagées :

- pgvector ou sqlite-vss ;
- chunking documentaire ;
- RAG hybride.

### 2. Agents spécialisés avancés

Agents à ajouter :

- Agent Recherche scientifique ;
- Agent Santé & veille médicale ;
- Agent Développement logiciel ;
- Agent Création contenu ;
- Agent Analyse documentaire.

Fonctions prévues :

- orchestration multi-agents ;
- collaboration entre agents ;
- validation croisée des réponses.

### 3. IA locale améliorée

Modèles à supporter :

- llama3 ;
- deepseek ;
- qwen ;
- mistral ;
- phi.

Fonctions prévues :

- téléchargement de modèles ;
- benchmark local ;
- sélection automatique du modèle ;
- monitoring RAM/VRAM.

### 4. Interface moderne

Améliorations prévues :

- multi-fenêtres ;
- workspace dynamique ;
- onglets ;
- thèmes personnalisés ;
- recherche globale ;
- raccourcis clavier.

### 5. Documents intelligents

Formats à supporter :

- PDF ;
- Word ;
- Excel ;
- Markdown ;
- images ;
- audio.

Fonctions prévues :

- résumé ;
- extraction ;
- OCR ;
- annotations IA.

### 6. Automatisation personnelle

Fonctions à créer :

- scheduler intégré ;
- tâches périodiques ;
- rappels ;
- workflows IA ;
- intégration n8n.

### 7. Multimodalité

Capacités à ajouter :

- STT ;
- TTS ;
- webcam ;
- vision IA ;
- capture écran ;
- génération image.

### 8. Sécurité

Fonctions à ajouter :

- chiffrement mémoire ;
- coffre de secrets ;
- audit logs ;
- sandbox des outils ;
- permissions agents.

### 9. Mobile & PWA

Améliorations prévues :

- mode offline mobile ;
- synchronisation ;
- notifications push ;
- optimisation Android.

### 10. Monitoring IA

Métriques à créer :

- métriques agents ;
- temps de réponse ;
- coûts ;
- usage mémoire ;
- logs IA.

## Futures releases

### v0.3.0

- orchestration autonome ;
- agents collaboratifs ;
- workflows visuels.

### v0.4.0

- assistant vocal temps réel ;
- mémoire contextuelle globale ;
